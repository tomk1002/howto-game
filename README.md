# HowTo

> 게임 안의 모든 "어떻게 하는 거야?"에 대한 답을, 직접 보여주는 도구.

숙련된 유저가 시연을 녹화하면, 다른 유저는 그 파일을 받아 자기 화면에서 단계별로 따라할 수 있는 데스크톱 앱.

---

## 무엇을 해결하나

게임의 깊은 기술·전략은 보통 유튜브 영상으로 배우게 됩니다. 그런데 영상은:

- 일시정지하고 따라하는 게 답답함 (스크럽·반복 재생)
- 화면 비율이 다르거나 UI가 바뀌면 매핑 안 됨
- 어느 시점에 어떤 키를 눌렀는지 정확히 모름

**HowTo** 는 입력 시퀀스 자체를 파일로 저장합니다. 받는 사람은 자기 화면에서 같은 시퀀스를 시각적으로 따라가며 배웁니다.

## 빠른 시작

```bash
git clone <repo>
cd HowTo
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt
python main.py
```

기본 사용:
1. 앱 실행 후 제목·게임 입력
2. **F9** 또는 [녹화] 버튼으로 녹화 시작 (다른 창이 포커스 잡고 있어도 작동)
3. 게임에서 시연
4. **F9** 다시 눌러 정지
5. [저장…] 으로 파일 저장
6. 친구에게 .json 파일 전달 → 친구가 [불러오기…] 로 열어 타임라인 확인

## 현재 기능

- ✅ 글로벌 단축키 (F9) 녹화 토글
- ✅ 키보드·마우스 클릭·스크롤 이벤트 + 타임스탬프 기록 (마우스 이동 제외)
- ✅ 타임라인 위젯 — 시간축 + 입력별 행 + 색상 구분
- ✅ JSON 저장/불러오기 (포맷 버전 관리)
- ✅ **게임에 입력 주입 안 함** — 안티치트 회피, 시각화·학습 목적

---

## 디자인 원칙

1. **안티치트 호환성 우선** — 입력을 게임에 주입하지 않고 "보여주기"만 함
2. **확장 가능한 포맷** — 같은 파일이 콤보·튜토리얼·풀 가이드까지 커버
3. **게임 비종속** — Windows OS 레벨에서 동작, 게임별 통합 0
4. **공유 친화** — 단일 .json 파일, 메일·디스코드로 공유 가능

## 데이터 포맷

```json
{
  "version": 1,
  "title": "Yasuo Q-cancel",
  "game": "League of Legends",
  "tags": ["mid", "mechanic"],
  "duration_ms": 1240,
  "created_at": "2026-04-27T05:00:00Z",
  "events": [
    { "t_ms": 0,    "type": "key_press",   "key": "q" },
    { "t_ms": 80,   "type": "mouse_press", "button": "Button.right", "x": 960, "y": 540 },
    { "t_ms": 95,   "type": "key_press",   "key": "s" },
    { "t_ms": 240,  "type": "key_release", "key": "q" }
  ]
}
```

장기 확장 슬롯: `steps[]` (단계 마커) · `annotations[]` (화살표·텍스트) · `audio_track` (내레이션) · `video_track` (참조 영상) · `branches[]` (조건부 경로).

## 기술 스택

- **Python 3.10+**
- **PyQt6** — UI
- **pynput** — 글로벌 입력 캡처

> 미래 단계 (M5+) 에서 게임 위 합성을 위해 일부 컴포넌트는 C# WPF + DirectComposition 또는 C++ 네이티브로 옮길 가능성 있음.

## 안티치트 정책

본 도구는 다음 원칙을 따릅니다:

- 게임 프로세스에 **주입하지 않음** (DLL 인젝션 X)
- 게임 메모리 **읽지 않음**
- 입력을 게임에 **전송하지 않음** (시각화만)
- OS 레벨 입력 캡처 (Raw Input) 만 사용

이는 Vanguard·Easy Anti-Cheat·BattlEye 등의 일반적인 거부 사유에서 벗어나도록 설계됩니다. 단, 특정 게임의 정책은 별도 협의가 필요할 수 있습니다.
